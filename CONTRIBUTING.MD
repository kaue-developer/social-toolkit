## 📜 Código de Conduta

Este projeto adota um [Código de Conduta](CODE_OF_CONDUCT.md). Ao participar, você concorda em mantê-lo.  

**Comportamentos inaceitáveis:** 
- Linguagem agressiva ou violenta
- Insultos e humilhações
- Assédio em todas as suas formas (sexual, moral, psicológico)
- Xingamentos e palavras de baixo calão direcionadas a pessoas
- Discriminação contra qualquer pessoa com base em gênero, raça, orientação sexual, religião, idade, deficiência, aparência ou origem

**Isso inclui, mas não se limita a:** piadas ofensivas, comentários depreciativos, apelidos pejorativos, microagressões ou qualquer forma de tratamento desrespeitoso.

## 🤝 Como posso contribuir?

### Tipos de contribuição:

#### 🐛 Reportar bugs
- Use o template de [Bug Report](.github/ISSUE_TEMPLATE/bug_report.md)
- Inclua: versão da API, endpoint afetado, erro retornado

#### 💡 Sugerir features
- Use o template de [Feature Request](.github/ISSUE_TEMPLATE/feature_request.md)
- Explique o caso de uso para redes sociais/plataformas

#### 📝 Melhorar documentação
- Correções no README
- Adicionar exemplos de uso da API
- Traduções

#### 🔧 Adição de novos utilitários
**Sistema de reações**:
- Likes/dislikes com tipos customizáveis

**Comentários de discussões**:
- Threads aninhados (resposta a resposta)
- Menções de usuários (@username)

**Sistema de chat**:
- Salas/grupos de chat
- Conversas diretas (DMs)

**Comunidades/grupos**:
- Criação de grupos/comunidades
- Regras personalizadas por comunidades/grupos
- Eventos
- Feed por comunidade/grupo
